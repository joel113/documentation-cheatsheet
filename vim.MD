# Vim Cheat Sheet

## Navigation

Move the cursor

     h
     j
     k
     l
     H
     M
     E

Jump forwards to the end of a word

     w

Jump forwards to the start of the a word

     b

Jump to the start of the line

     0

Jump to the end of the line

     $

Goto end of file

     Shift + G
